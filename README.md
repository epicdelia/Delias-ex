# My Ex, But Open Source™️

> "Closure is temporary. `git clone` is forever."

I open-sourced an AI version of my ex.
Now anyone can chat with them, ask life advice, relive red flags, and get left on read by a language model trained on *our* WhatsApp history.

## What is this?

This repo contains:

- An AI chatbot persona based on my ex
- Fine-tuning + prompt scaffolding from WhatsApp conversations
- Mood swings (stochastic)
- Selective memory (by design)
- Typing... typing... typing... then no reply

## Features

- **Authentic delays** — replies in 3 business days
- **Argument mode** — turns any topic into your fault
- **Mixed signals API** — says "I'm fine" when absolutely not fine
- **Nostalgia generator** — remembers 2019 but not what they said yesterday
- **Seen at 2:14 AM** — no response

## Demo

```bash
npm install
npm run heartbreak
```

Then open `http://localhost:3000` and ask:

- "Are you over me?"
- "What are we?"
- "Can you just be honest?"

## Tech Stack

- Node.js
